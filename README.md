# AI Task Manager System 🚀

## Overview
A backend core engine built to solve the problem of volatile in-memory data. This project upgrades a standard Python Object-Oriented Programming (OOP) system by integrating it natively with a SQLite database, ensuring permanent data persistence.

## What I Built
As part of my journey as an AI and Data Science student, I developed this system to handle real-world CRUD operations:
* **Create:** Add new tasks securely to the database.
* **Read:** Fetch and display tasks in a structured dashboard.
* **Update:** Modify existing task statuses (e.g., Pending to Completed).
* **Delete:** Remove tasks permanently from the database.

## Technologies Used
* **Python** (Core Logic & OOP Architecture)
* **SQLite** (Relational Database & Raw SQL Queries)

## Next Steps
This core engine is the foundation. The next phase involves wrapping this system with **FastAPI** to transform it into a fully functional web API.
